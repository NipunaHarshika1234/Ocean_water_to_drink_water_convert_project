Water Tank Monitoring and Automatic Motor Control System
📌 Overview

This project is an Arduino-based Smart Water Tank Monitoring and Automatic Motor Control System. It monitors water levels in two tanks using ultrasonic sensors, measures environmental conditions using a DHT22 sensor, reads water temperature using a DS18B20 sensor, and automatically controls a water pump through a relay module.

The system helps prevent water shortages, reduces manual monitoring, and improves efficient water management.

🚀 Features
💧 Dual Water Tank Level Monitoring using HC-SR04 Ultrasonic Sensors
🔄 Automatic Water Pump ON/OFF Control
🌡️ Ambient Temperature Monitoring using DHT22
💦 Humidity Monitoring using DHT22
🌡️ Water Temperature Monitoring using DS18B20
🚨 Error Detection and LED Indication
💡 Motor Status Indicator LED
📟 Real-Time Serial Monitor Output
⚡ Automatic Tank Refill System
